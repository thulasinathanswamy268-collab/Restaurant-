# Restaurant Menu System

A full-stack restaurant menu application built with Flask, Supabase, and a local chatbot.

## Features
- Display menu items from Supabase
- Search functionality
- Shopping cart
- Place orders
- Local chatbot for menu/website questions
- Responsive design with Tailwind CSS

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Set up Supabase project and add credentials to `.env`
3. Create tables: `menu_items` and `orders`
4. Run: `python app.py`

## Deployment
Deploy to Render.com or Railway for free hosting.
